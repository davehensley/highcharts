Bubble series

A bubble series is a three dimensional series type where each point renders an X, Y and Z value. Each points is drawn as a bubble where the position along the X and Y axes mark the X and Y values, and the size of the bubble relates to the Z value.

Bubble size

The size of the bubbles is controlled by the minSize and maxSize options. These options can be given as either pixels or percentages, where the percentage relates to the plot area size. When multiple bubble series are present in the same chart, the bubble sizes are computed from the combined points. 

	minSize: 8,    // pixel value
	maxSize: '20%' // percentage value

Bubble color

By default, the bubbles' line (border) color is inherited from the series.color option, and the fill is the series.color with a default opacity of 0.5. These values ensure that overlapping bubbles are clearly visible. For customization the lineColor and fillColor can be set independently.

A common application of the fill color is to apply a radial gradient to give the impression of 3D bubbles floating in space. This can be done by setting the fillColor option to a radial gradient: 

fillColor: {
    radialGradient: { cx: 0.4, cy: 0.3, r: 0.7 },
    stops: [
        [0, 'rgba(255,255,255,0.5)'],
        [1, 'rgba(69,114,167,0.5)']
    ]
}
See radial gradients demonstrated.

Data labels

By default, data labels for bubble series are placed on top of the bubbles by setting the align option to "center", verticalAlign to "middle" and inside to true. In the dataLabels.style option, the labels are given a default color of "white", and in modern browsers a black text shadow to make it stand out from the background.
